# Server for COVID-19 Italy Data

## Description

Server Proxy Api for extract data to [Dati COVID-19 Italia](https://github.com/pcm-dpc/COVID-19)

This Project is develop with [NestJS](http://nestjs.com/)

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
